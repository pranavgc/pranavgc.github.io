# Pranav Chandratre — Portfolio Website

Personal portfolio website for **Pranav Ganesh Chandratre**, MSc Data Science student at the University of Birmingham and machine learning researcher.

🌐 **Live site**: [pranavgc.github.io](https://pranavgc.github.io)

---

## About

A single-page portfolio showcasing my background in physics-driven machine learning, deep learning research, and applied data science. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies beyond Google Fonts.

---

## Sections

- **Hero** — Introduction with animated neural network canvas
- **About** — Background, education, and key stats
- **Skills** — Technical expertise grouped by domain
- **Projects** — Selected work including GNN research and ML systems
- **Certifications** — Courses and industry credentials
- **CV** — Downloadable curriculum vitae
- **Contact** — Links to email, LinkedIn, GitHub, and HuggingFace

---

## Tech Stack

| Layer | Details |
|---|---|
| Markup | HTML5 |
| Styling | CSS3 (custom properties, CSS Grid, Flexbox) |
| Scripting | Vanilla JavaScript (Canvas API, IntersectionObserver) |
| Fonts | Google Fonts — Cormorant Garamond, IBM Plex Mono, Outfit |
| Hosting | GitHub Pages |

No build step. No npm. Just open `index.html`.

## Local Development

```bash
git clone https://github.com/pranavgc/pranavgc.github.io.git
cd pranavgc.github.io
# Open index.html in your browser — or use a local server:
npx serve .
```

---

## Deployment

This site is deployed via **GitHub Pages** from the `main` branch root.

To deploy your own fork:
1. Fork this repository
2. Rename it to `<your-username>.github.io`
3. Go to **Settings → Pages → Source** and select `main` / `root`
4. Your site will be live at `https://<your-username>.github.io`
